## ⚙️ Kurulum
1. Bu repoyu klonlamak için:
  ```bash
    https://github.com/EnsAkyl/flutter-todo-app.git
  ```
2. Gerekli bağımlılıkları yükleyin: **npm install**
3. Uygulamayı Başlatın: **npm start**
